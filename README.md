# CogGRAG
- 一种分为三阶段的GRAG：自顶向下的问题分解、层次化知识检索、带有自我验证的推理生成
- 传统的知识图谱问答方法多采用单步检索或固定模板匹配的方式，在面对涉及多重关联、隐含语义与长程依赖的复杂非遗文化查询时，往往因推理深度不足而难以给出准确、完整的答案。检索增强生成（RAG）范式通过将外部知识引入大语言模型的生成过程，在一定程度上缓解了知识覆盖不足的问题，但基于向量相似度的检索机制在处理非遗知识图谱中高度关联的实体与关系时仍存在明显局限：一方面，碎片化的知识片段难以保留传承谱系、技法演变等结构性语义；另一方面，缺乏全局推理规划的迭代式检索容易造成误差累积，影响最终答案的可靠性。
- 针对上述问题，本研究借鉴人类解决复杂问题时的认知策略，参考Yao Cheng等提出的认知启发式GRAG（简称CogGRAG）框架的设计思想，构建面向非物质文化遗产知识图谱的CogGRAG图谱增强检索与生成框架。该框架模拟人在面对陌生问题时“先整体规划、再分步求证、最后反思校验”的思维过程，将推理任务分解为三个协同环节：自顶向下的问题分解、层次化知识检索以及带有自我验证的推理生成。
